# APUF – Associação dos Trabalhadores Urbanos de Fernandópolis

A APUF é uma plataforma web desenvolvida para apoiar os trabalhadores urbanos de Fernandópolis, oferecendo um sistema moderno para divulgação de produtos, controle de pedidos e fortalecimento da economia local por meio da tecnologia.

## Funcionalidades

- Exibição de produtos e serviços
- Sistema de pedidos
- Área de login para usuários e administradores
- Gerenciamento de produtos
- Gerenciamento de usuários
- Controle de vendas
- Carrinho de compras
- Interface responsiva

## Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  

## Objetivo do Projeto

O objetivo da APUF é digitalizar e facilitar a organização dos trabalhadores urbanos, oferecendo uma solução acessível, prática e moderna para vendas e gestão, promovendo inclusão digital e fortalecimento da comunidade local.
